# OMinimoScrapper

A web scrapper to get links to articles from a web site given the article titles.

The articles are parsed to extract some metadata.

The scrapper produces a JSON file with links and processed metadata.

## Technologies

- C#
- .Net 6
- Async/Await
- JSON
- Regular Expressions
- HtmlAgilityPack
